# sokoban-web

A lightweight Sokoban game built with plain HTML, CSS, and JavaScript.

## Features

- Three playable levels
- Timer and move counter
- Keyboard and on-screen direction controls
- In-page level clear modal
- Ready for GitHub Pages deployment

## Local Preview

```powershell
cd E:\mypro\sokoban-web
node .\dev-server.js
```

Open `http://127.0.0.1:4173` in your browser.

## Deploy

This project is set up for GitHub Pages deployment through GitHub Actions.
After enabling Pages with `GitHub Actions`, pushes to `main` will publish the site.
